# Documentation for Public Transport Optimizer

## Tech Stack
- **Backend:** FastAPI, Python
- **Frontend:** React + Vite
- **Libraries:** NetworkX, Folium, OSRM, Pandas

## Project Structure
- `app/`: Contains the FastAPI backend
- `frontend/`: Contains the React frontend
- `docs/`: Documentation files
- `requirements.txt`: Python dependencies
- `package.json`: JavaScript dependencies

## API Endpoints
- **GET /api/routes**: Retrieves available routes
- **POST /api/routes**: Creates a new route
- **GET /api/transport/{id}**: Retrieves transport details by ID
